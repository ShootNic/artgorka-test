## Краткая инструкция по работе
Для начала работы у вас должен быть установлен **Node.js** 12 версии

nmv install 12.22.7

nmv use 12.22.7

Установка - `npm i`

### Очистка папки dist
```
npm run clear
```

### Запуск сервера для разработки
```
npm run dev
```

### Сборка проекта
```
npm run build
```
